# SoundBoost Pro - User Manual

## Getting Started

### First Launch

When you first launch SoundBoost Pro, you'll see the main interface with:
- **Navigation Panel** (left side): Access to different views
- **Master Controls** (top): Volume, bypass, and preset selection
- **Content Area** (center): Current view content
- **Status Bar** (bottom): Application status

### Quick Start

1. **Select Audio Device**: Go to Settings > Audio and choose your input/output devices
2. **Choose a Preset**: Select from the preset dropdown (Bass Boost, Rock, Jazz, etc.)
3. **Adjust Volume**: Use the master volume slider
4. **Enable/Disable**: Toggle the master power button

## Views

### Home View
The main dashboard showing:
- Real-time spectrum analyzer
- VU meters for left/right channels
- Quick access to common controls

### Equalizer View
10-band parametric equalizer:
- **Frequency Bands**: 31Hz, 62Hz, 125Hz, 250Hz, 500Hz, 1kHz, 2kHz, 4kHz, 8kHz, 16kHz
- **Adjustment**: Click and drag each band up/down
- **Presets**: Flat, Bass Boost, Treble Boost, V-Shape, Vocal Boost, etc.

### Bass Boost View
Enhance low frequencies:
- **Frequency**: Center frequency (20-200 Hz)
- **Gain**: Boost amount (0-18 dB)
- **Width**: Bandwidth/Q factor
- **Harmonics**: Add harmonic richness
- **Sub-bass**: Enable sub-bass synthesis

### Effects View
Manage your effect chain:
- **Noise Gate**: Remove background noise
- **Compressor**: Control dynamic range
- **Stereo Widener**: Enhance stereo image
- **Reverb**: Add spatial effects
- **Limiter**: Prevent clipping

### Analyzer View
Detailed audio analysis:
- **Spectrum**: Real-time frequency display
- **Waveform**: Time-domain visualization
- **Level Meters**: Peak and RMS levels
- **Frequency Analysis**: Detailed frequency data

### Settings View
Configure the application:
- **Audio**: Sample rate, buffer size, devices
- **UI**: Theme selection, minimize behavior
- **ML**: Enable/disable ML features

### ML Insights View
AI-powered audio analysis:
- **Genre Detection**: Identifies music genre
- **BPM Detection**: Shows beats per minute
- **Voice Detection**: Indicates vocal presence
- **Adaptive EQ**: Suggests EQ settings

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+M | Toggle mute |
| Ctrl+B | Toggle bypass |
| Ctrl+1-7 | Switch views |
| Ctrl+Q | Quit |
| F1 | Help |

## Tips

1. **Start with presets**: Use presets as starting points, then fine-tune
2. **Watch your levels**: Keep peaks below 0 dB to avoid clipping
3. **Use spectrum analyzer**: Visual feedback helps identify problem frequencies
4. **ML Insights**: Use genre detection for automatic EQ suggestions
5. **Save your presets**: Create custom presets for different content types
